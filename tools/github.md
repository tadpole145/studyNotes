

## Github的使用介绍

### github搜索技能

github条件搜索技能小结

- 搜索名字   in:name xxx
- 搜索描述   in: description xxx
- 搜索readme   in: readme xxx
- 按 stars       stars: >2000
- 按fork          fork:>3000
- 仓库大小搜索     size:>=5000  [说明5000大小是5000k]
- 按更新时间 pushed:>20200101
- 按语言           language: xxx
- 按作者名       user:xxx[注意:后面不要有空格]

搜索的方式可以组合
